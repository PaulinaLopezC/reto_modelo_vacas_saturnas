## Modelo primera iteración (PREDICCIÓN PARA EL PERIODO DE SECADO DE LAS VACAS)

Para esta entrega se puede observar en el documento que tenemos la primera iteración tomamos en cuenta los valores como time-series, para poder tratar de rpedecir el comportamiento final de las lactancias de cada baca basandomos en el comportamiento inicial de su misma lactancia. Al final de esto modelo y de su evaluación se determino que esta enfoque hacia los datos no fue el adecuado debido a que como las vacas son demasiado diferentes en cada una de las lactancias seria necesario crear un modelo LSTM para cada una de las vacas. Debido a esto se decidio buscar otro enfoque llendo con la doctora para saber como ellos determinan a travez de los datos el momento optimo de secado de cada una de las vacas.
